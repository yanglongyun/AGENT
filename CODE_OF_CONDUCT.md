# 贡献者行为准则

## 我们的承诺

我们承诺为每一位参与者提供一个无骚扰的社区环境，不因年龄、体型、残障、种族、性别特征、性别认同与表达、经验水平、教育背景、社会经济状况、国籍、外貌、肤色、宗教或性取向而区别对待。

## 期望的行为

- 展现同理心与善意。
- 尊重不同的观点和经历。
- 给予并接受有建设性的反馈。
- 承担责任，在犯错时道歉并学习。
- 以社区整体利益为重。

## 不可接受的行为

- 性化语言、图像或不受欢迎的性关注。
- 挑衅、侮辱、贬损性评论或人身、政治攻击。
- 公开或私下骚扰。
- 未经许可公布他人住址、邮箱等私人信息。
- 在专业场景中合理认定为不当的其他行为。

## 执行

项目维护者负责解释并执行本准则，可删除、编辑或拒绝不符合准则的 Issue、Pull Request、评论、提交和其他贡献。对于骚扰、滥用或其他不可接受行为，维护者可根据影响采取纠正、警告、临时禁止或永久禁止等措施。

报告问题时，请通过 Gitee 私信联系 [@realuckyang](https://gitee.com/realuckyang)。如无法私信，可创建一个不含敏感细节的 Issue，请维护者主动联系。所有报告将被尊重并尽可能保密处理。

## 适用范围与归属

本准则适用于项目空间，以及个人在公开场合代表项目或社区时。

本准则改编自 [Contributor Covenant 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct.html)。
